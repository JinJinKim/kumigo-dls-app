경북 구미시에 위치한 구미고등학교의 도서관을 쉽게 이용할 수 있는 어플리케이션입니다.

기능
==========
- 장서 검색
- 인기 도서 목록

설명
==========
2015년 6월에 개발하여 출시까지 완료된 어플리케이션입니다.
경상북도교육청의 DLS를 이용하여 컨텐츠를 제공합니다.

사용된 라이브러리
==========
- Jericho HTML Parser
- [Android-studio-material-template](https://github.com/kanytu/Android-studio-material-template)
